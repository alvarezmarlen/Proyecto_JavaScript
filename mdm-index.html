Métodos de Modificación

Leer, Modificar y Eliminar Atributos HTML con JavaScript

Con los métodos getAttribute(), setAttribute() y removeAttribute() podremos trabajar con atributos HTML.

Leer Atributos getAttribute()

Acceder al valor original de un atributo tal como fue escrito en el HTML.

Por ejemplo,

<a id="link" href="https://ejemplo.com" target="_blank">Ir a ejemplo</a>

Podemos obtener sus atributos usando,
JavaScript:

const enlace = document.getElementById("link");
const url = enlace.getAttribute("href"); // "https://ejemplo.com"
const target = enlace.getAttribute("target"); // "_blank"

Mostrar el src de una imagen.

Modificar o agregar Atributos SetAttribute()

Cambiar el valor de un atributo existente o añadir uno nuevo.
Nombre del atributo y el valor que queremos asignar.

Un ejemplo seria para modificar atributos existentes,

<img id="logo" src="logo.png" alt="Logo anterior">

En JavaScript,

const logo = document.getElementById("logo");
logo.setAttribute("alt", "Logo actualizado");

Cambiar imagen.

Eliminar elementos con RemoveAttribute()

Elimina un atributo completamente del HTML cuando ya no es necesario

Por ejemplo,

<input id="correo" type="email" required>

Si luego deseas permitir enviar el formulario sin ese campo, puedes eliminar el atributo requied, así:

const campo = document.getElementById("correo");
campo.removeAttribute("required");

Eliminar un atributo no solo cambia el HTML, sino que también puede impactar directamente en el comportamiento del elemento.

Activar un input quitando el atributo disabled.

Muestra un booleano hasAttribute()

Por ejemplo,

<p id="texto" class="importante">Hola</p>

En JavaScript,

let p = document.getElementById("texto");
console.log(p.hasAttribute("class")); // true
console.log(p.hasAttribute("style")); // false

Tiene la clase → true
No tiene el atributo style → false

Modificar estilos y CSS del DOM con javascript

Modificcion de los estilos, tanto directa, o por clase.

Modificación directa de estilos CSS

Utilizando la propiedad style el elemento y asignando valores a las propiedades CSS.

Comando = elemento.style.propiedadCSS = 'valor'

Ejemplo,

<div id="caja">Hola mundo</div>

const caja = document.getElementById("caja");
caja.style.color = "white";
caja.style.backgroundColor = "blue";
caja.style.padding = "20px";
caja.style.fontSize = "24px";

Manipulación de Clases CSS

Utilizamos las propiedades classList del elemento para agregar, eliminar o alternar clases.

add()

Añade una o más clases a un elemento.
Por ejemplo,

<div id="caja">Soy una caja</div>
<button id="btnAdd">Agregar clase</button>

En Css,

.rojo {
background: red;
color: white;
padding: 10px;
}

En JavaScript,

const caja = document.getElementById("caja");
const btnAdd = document.getElementById("btnAdd");

btnAdd.addEventListener("click", () => {
caja.classList.add("rojo");
});

remove()

Elimina una o más clases de un elemento.
Por ejemplo quitar una clase de un elemento,

<div id="caja" class="activo">Caja con clase inicial</div>
<button id="btnRemove">Quitar clase</button>

En Css,

.activo {
border: 3px solid green;
padding: 10px;
}

En JavaScript,

const caja = document.getElementById("caja");
const btnRemove = document.getElementById("btnRemove");
btnRemove.addEventListener("click", () => {
caja.classList.remove("activo");
});

toggle()

Si la clase no está, la agrega, y si ya está, la quita.
Por ejemplo,

<div id="caja">Pasa el mouse o haz clic</div>
<button id="btnToggle">Alternar clase</button>

En Css,

.resaltado {
background: yellow;
border: 2px solid orange;
transform: scale(1.1);
transition: 0.2s;
}

En JavaScript,

const caja = document.getElementById("caja");
const btnToggle = document.getElementById("btnToggle");
btnToggle.addEventListener("click", () => {
caja.classList.toggle("resaltado");
});
