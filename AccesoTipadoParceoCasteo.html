JavaScript es un lenguaje de tipado dinámico, Las variables en JavaScript no están asociadas directamente con ningún tipo de valor en particular, y a cualquier variable se le puede asignar (y reasignar) valores de todos los tipos:


  • let foo = 42; // foo ahora es un número
  • foo = "bar"; // foo ahora es un string
  • foo = true; // foo ahora es un booleano

Notacion ()

La notación () en JavaScript se usa principalmente para la llamada a funciones, la agrupación de expresiones para controlar la precedencia de operadores, y en el operador condicional ternario (? :). También aparece en la definición de parámetros de función y en la sintaxis de las clases para la herencia o el constructor

Usos principales

  • Llamada a funciones: Se utilizan los paréntesis () para invocar (ejecutar) una función.
  • miFuncion();
  • Agrupación de expresiones: Permiten forzar un orden específico en la evaluación de operaciones matemáticas, agrupando sub-expresiones.
  • let resultado = (5 + 3) * 2; // El resultado es 16
  • Operador condicional ternario: Es el único operador que requiere tres operandos: una condición, una expresión a ejecutar si la condición es verdadera, y una expresión a ejecutar si es falsa.
  • let mensaje = (edad >= 18) ? "Mayor de edad" : "Menor de edad";
  • Definición de parámetros: Al declarar una función, los paréntesis encierran los parámetros de entrada.
  • function sumar(a, b) { return a + b; }

Caracteres de Escape

Los caracteres de escape en JavaScript se usan para representar caracteres especiales que no se pueden escribir directamente en una cadena, usando la barra invertida (\(``\`{`}\)) seguida de otro carácter. Ejemplos comunes son $``\\n``$ para una nueva línea, $``\\t``$ para una tabulación o $``\\``$ para mostrar una barra invertida literal

Secuencias de escape comunes

  • ``\\``': Barra invertida para escapar un carácter de barra invertida.
  • ``\\``": Barra invertida para escapar comillas dobles dentro de una cadena de comillas dobles.
  • ``\\``': Barra invertida para escapar comillas simples dentro de una cadena de comillas simples.
  • ``\\n``$: Nueva línea.
  • ``\\t``$: Tabulación.
  • ``\\b``$: Retroceso.
  • ``\\f``$: Salto de página.
  • ``\\f``$: Salto de página.

Parseo/Casteo

Tipos de variable de entrada

En JavaScript, los tipos de variable de entrada se pueden agrupar enprimitivos y objetos. Los tipos primitivos son number, string, bigint, boolean, null, undefined y symbol. El tipo no primitivo es object, que incluye objetos complejos como arreglos (arrays), funciones y objetos personalizados

Tipos primitivos

  • number: Para números enteros y decimales.
  • string: Para secuencias de caracteres (texto).
  • bigint: Para números enteros más allá del rango estándar de number.
  • boolean: Para valores lógicos de true o false.
  • null: Representa la ausencia intencionada de valor.
  • undefined: Representa un valor que no ha sido asignado.
  • symbol: Un valor único e inmutable.

Tipos no primitivos

object: Un tipo complejo que puede contener colecciones de datos y estructuras de mayor complejidad. Incluye:

  • Arrays: Listas de valores.
  • function: Que pueden ser asignadas a variables.
  • Objetos personalizados: Estructuras con propiedades y métodos.

Parseo

significa analizar y convertir una cadena de texto en un formato que el motor de JavaScript pueda entender Los métodos de "parseo" más comunes son:

  • JSON.parse() para convertir una cadena en formato JSON a un objeto JavaScript,
  • parseInt() para convertir una cadena a un número entero
  • parseFloat() para convertir una cadena a un número decimal
  • Date.parse() para analizar una cadena y convertirla en el número de milisegundos de una fecha.
  • El String()El método convierte un valor en una cadena.

Casteo

es el proceso de convertir un valor de un tipo de dato a otro. Se le conoce más formalmente como conversión de tipos.

1. Conversión implícita (coerción de tipos)

Esta conversión ocurre automáticamente cuando JavaScript realiza una operación entre valores de tipos de datos diferentes. El intérprete intenta convertir uno de los valores para que la operación pueda ser completada.

Ejemplos

Al sumar una cadena de texto y un número, JavaScript convierte el número a cadena para concatenarlos.

let resultado = '5' + 10; // El resultado es la cadena "510"

1. Conversión explícita

Esta conversión se realiza manualmente, usando funciones o métodos para cambiar el tipo de dato de forma intencional. Esto es útil para asegurarse de que el código se comporte como se espera y evitar errores.

Convertir a número: Usando las funciones Number(), parseInt() o parseFloat().

let texto = '25';
let numero = Number(texto); // numero es el número 25
